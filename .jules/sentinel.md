## 2026-05-14 - Improved Terminal Injection Protection
**Vulnerability:** Terminal injection and spoofing via ANSI escape sequences (OSC) and malicious control characters.
**Learning:** Standard ANSI regexes often miss OSC (Operating System Command) sequences, which can be used to rename terminal titles or hide malicious commands. Additionally, overly broad control character stripping can remove legitimate formatting (like newlines and tabs), while missing "Trojan Source" BiDi overrides.
**Prevention:** Use a comprehensive ANSI regex that includes support for CSI and OSC sequences. When stripping control characters, use targeted ranges to preserve safe whitespace (`\n`, `\t`) while specifically targeting dangerous C0/C1 characters and Unicode BiDi control characters (`\u202a-\u202e`, `\u2066-\u2069`).
