# OLLAMA REMOTE SERVER SCANNER

A lightweight, friendly tool for discovering and inspecting remote Ollama servers on your network. This README is intentionally non-technical — see **GUIDE.md** for full installation steps, usage examples, and advanced options.

---

🌈 ## Overview

**Ollama Remote Server Scanner** helps you quickly locate Ollama instances on a network and gather basic metadata such as version and available models. It’s designed for developers and operators who want a fast, non-intrusive way to inventory Ollama servers.

---

🤖 ## Key Features

- **Discovery** of Ollama servers on local and remote networks  
- **Metadata collection** including version and model lists  
- **Human-friendly output** suitable for quick triage and automation  
- **Exportable results** for reporting and integration

---

⚙️ ## Quick Start
 
1. Open **GUIDE.md** for exact installation and run instructions specific to this project.  
2. Follow the commands in **GUIDE.md** to run discovery and scanning.

---

## Usage Examples

- **Discover on local network** — run the discovery workflow described in **GUIDE.md** to list reachable Ollama servers.  
- **Inspect a single host** — see the host-targeting examples in **GUIDE.md** for how to query a specific IP or hostname.  
- **Export results** — the guide documents available output formats (JSON, table, plain text) and how to save results.

## Configuration

Configuration options such as network ranges, timeouts, ports, and output formats are documented in **GUIDE.md**. If your environment requires authentication, follow the authentication section there.

---

⚡ ## Security and Responsible Use

- The scanner performs **non-destructive** metadata queries.  
- Only scan networks and hosts you own or have explicit permission to scan.  
- Do not run scans across untrusted networks without authorization.
